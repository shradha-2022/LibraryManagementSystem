A simple **Library Management System** built using **Java**, **Spring Boot**, and **Maven**, featuring book, author, and category management with an admin interface.

## üöÄ Local Setup

Follow these steps to run the project locally:

1. **Clone the repo**:

```bash
git clone https://github.com/YOUR_USERNAME/LibraryManagementSystem.git
cd LibraryManagementSystem
```

2. **Install JDK 17**
   [Download JDK 17](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html)

3. **Install an IDE**: IntelliJ IDEA, Eclipse, or NetBeans

4. **Install Apache Maven**
   [Installation Guide](https://maven.apache.org/install.html)

5. **Build the project**:

```bash
mvn clean install
```

6. **Run the project**:

```bash
mvn spring-boot:run
```

7. **Open in browser**:
   Visit [http://localhost:9080](http://localhost:9080)

8. **Admin Login Credentials**:

* **Email**: `admin@admin.in`
* **Password**: `Temp123`
